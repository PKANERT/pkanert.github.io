KANERT Robot Tools

This project is all about building a handy toolkit for industrial robot integrators. It will covers things like collision matrix, safety configuration, customer standards, and visualizing data from robot backups.

It will be completely free, open-source, and built with the idea that anyone can jump in and help it grow.

Currently in development:

Collision Matrix

Collision analysis between KUKA robots.

CellSpace Assistant

Configuration and visualization of KUKA robot workspaces.

Safety XML Reader

Reading and comparing XML data from KUKA backups.

Standard IW7

Checking robot programs compliance with the Daimler IW7 standard.
